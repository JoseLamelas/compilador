O que é JavaScript?

JavaScript é uma linguagem que torna as páginas web interativas e dinâmicas. Imagine um site como uma casa: o HTML é a estrutura (paredes e telhado), o CSS é a decoração (cores e móveis), e o JavaScript é a eletricidade que faz as luzes acenderem, os eletrodomésticos funcionarem e as portas abrirem automaticamente. Com JavaScript, você pode criar botões que reagem a cliques, formulários que validam dados, ou até jogos diretamente no navegador!

Originalmente, o JavaScript foi criado para funcionar nos navegadores (como Chrome ou Firefox), mas hoje ele é usado em servidores (com Node.js), aplicativos móveis, e até dispositivos inteligentes, como lâmpadas conectadas. É uma linguagem fácil de começar, mas poderosa o suficiente para construir sistemas complexos.

Por que aprender JavaScript?

  • Interatividade: Faz sites responderem às ações dos usuários, como clicar ou rolar a página.
  • Versatilidade: Usado em sites, aplicativos, servidores e mais.
  • Comunidade: Tem uma enorme comunidade e muitas bibliotecas prontas, como React para interfaces modernas.

Onde o JavaScript é usado?

  • Frontend: Sites interativos, como menus que abrem ou sliders de imagens.
  • Backend: Servidores que processam dados, como em lojas online.
  • Mobile: Apps como Uber, criados com React Native.
  • Desktop: Programas como Visual Studio Code, feitos com Electron.
  • Jogos: Jogos simples no navegador ou até em plataformas como itch.io.

Sintaxe e Conceitos Básicos

A sintaxe do JavaScript é como a gramática de uma língua: são as regras para escrever código que o computador entende. Aqui, você aprenderá os conceitos básicos, como guardar informações, criar funções, tomar decisões e repetir ações.

Variáveis e Tipos de Dados

Variáveis são como potes onde você guarda informações, como nomes, números ou listas. Em JavaScript, usamos let para valores que podem mudar, const para valores fixos, e var (menos comum). Cada dado tem um tipo, como texto (string) ou número.

Por que isso é útil? Imagine que você está criando um site de cadastro: variáveis guardam o nome e a idade do usuário para usá-los depois.

Como funciona: O código cria variáveis com diferentes tipos de dados e usa typeof para mostrar o tipo de cada uma. O resultado é exibido no elemento <div id="output">, que está no HTML. O método innerHTML insere o texto formatado no DOM.


let nome = "João";           // String: texto entre aspas
const idade = 25;            // Number: número sem aspas
var ativo = true;            // Boolean: verdadeiro ou falso
let hobbies = ["leitura", "programação"]; // Array: lista de itens
let pessoa = {               // Object: conjunto de propriedades
  nome: "Maria",
  idade: 30,
  profissao: "Desenvolvedora"
};

// Exibir tipos de dados no elemento output
let output = document.getElementById("output");
output.innerHTML = `
  Tipo de nome: ${typeof nome}

  Tipo de idade: ${typeof idade}

  Tipo de ativo: ${typeof ativo}

  Tipo de hobbies: ${typeof hobbies}

  Tipo de pessoa: ${typeof pessoa}
`;

Funções

Funções são como receitas: você define os passos uma vez e pode usá-los várias vezes. Elas ajudam a organizar o código e realizar tarefas específicas, como dizer "Olá" ou calcular algo.

Por que isso é útil? Se você tem um site que mostra saudações para cada visitante, uma função evita repetir o mesmo código.

Como funciona: O código define três tipos de funções: tradicional, arrow, e anônima. Cada uma realiza uma tarefa (saudação, soma, multiplicação) e o resultado é mostrado no <div id="output">. A função saudacao usa um parâmetro nome, enquanto somar e multiplicar usam dois parâmetros.


function saudacao(nome) {
  return "Olá, " + nome + "!"; // Concatena o nome com uma saudação
}

const somar = (a, b) => a + b; // Soma dois números

const multiplicar = function(a, b) {
  return a * b; // Multiplica dois números
};

// Exibir resultados no elemento output
let output = document.getElementById("output");
output.innerHTML = `
  Saudação: ${saudacao("Pedro")}

  Soma: ${somar(5, 3)}

  Multiplicação: ${multiplicar(4, 6)}
`;

Estruturas de Controle

As estruturas de controle decidem o que o código faz com base em condições (como "se isso, faça aquilo") ou repetem ações (como "faça isso 5 vezes"). Elas são essenciais para criar lógica no seu programa.

Por que isso é útil? Em um site de ingressos, você pode usar condições para verificar se alguém é maior de idade ou loops para listar eventos disponíveis.

Como funciona: O código usa if/else para verificar a idade, switch para escolher um dia, e diferentes loops (for, for...of, for...in, while) para repetir ações. Cada resultado é concatenado em variáveis e exibido no <div id="output"> com quebras de linha (<br>) para clareza.

} else if (idade >= 16) {
  resultado = "Pode conduzir";
} else {
  resultado = "Menor de idade";
}

let diaSemana = 1;
let dia = "";
switch (diaSemana) {
  case 1:
    dia = "Segunda-feira";
    break;
  case 2:
    dia = "Terça-feira";
    break;
  default:
    dia = "Outro dia";
}

let números = "";
for (let i = 0; i < 5; i++) {
  números += "Número " + i + "
";
}

let hobbies = ["leitura", "programação"];
let listaHobbies = "";
for (let hobby of hobbies) {
  listaHobbies += hobby + "
";

  propriedades += propriedade + ": " + pessoa[propriedade] + "

${números}
  For...of:
${listaHobbies}

${propriedades}
  While:
${contagem}

Arrays e Métodos

Arrays são listas que guardam vários valores, como uma lista de compras. JavaScript tem métodos para adicionar, remover ou transformar itens nos arrays.

Por que isso é útil? Em um site de loja, você pode usar arrays para listar produtos, filtrar os que estão em promoção, ou calcular o total do carrinho.

Como funciona: O código cria um array e usa métodos como push, pop, filter, map, e reduce para manipulá-lo. O método join transforma arrays em texto com vírgulas. Os resultados são exibidos no <div id="output">.


let números = [1, 2, 3, 4, 5];

números.push(6);              // Adiciona 6 ao final
números.unshift(0);           // Adiciona 0 ao início
números.pop();                // Remove o último (6)
números.shift();              // Remove o primeiro (0)

let pares = números.filter(n => n % 2 === 0); // Filtra números pares
let dobrados = números.map(n => n * 2);       // Dobra cada número
let soma = números.reduce((acc, n) => acc + n, 0); // Soma todos

let temTres = números.includes(3); // Verifica se tem 3
let índiceDois = números.indexOf(2); // Encontra o índice do 2
let encontrado = números.find(n => n > 3); // Encontra o primeiro > 3

let output = document.getElementById("output");
output.innerHTML = `
  Pares: ${pares.join(", ")}

  Dobrados: ${dobrados.join(", ")}

  Soma: ${soma}

  Tem 3: ${temTres}

  Índice do 2: ${indiceDois}

  Encontrado > 3: ${encontrado}
`;

Manipulação do DOM

O DOM (Document Object Model) é como um mapa da sua página HTML. Ele transforma os elementos (como botões e textos) em objetos que o JavaScript pode mudar, criando interatividade.

Por que isso é útil? Você pode usar o DOM para mudar o texto de um título, esconder um menu, ou criar novos itens em uma lista de tarefas.

Antes de mudar algo na página, você precisa selecionar o elemento. JavaScript oferece métodos como getElementById ou querySelector para encontrar elementos.

Como funciona: O código captura o envio de um formulário, impede o recarregamento com preventDefault, e valida os campos. O evento input verifica o email em tempo real, mudando a borda do campo. Os resultados são exibidos no <div id="output">.

Email: ${email}`;

Programação Assíncrona

JavaScript pode esperar por tarefas que levam tempo, como carregar dados de um servidor, sem travar a página. Isso é feito com callbacks, Promises, ou async/await.

Por que isso é útil? Em um site de redes sociais, você pode carregar posts enquanto o usuário continua navegando.

Callbacks

Callbacks são funções passadas como argumentos para outras funções, executadas quando a tarefa termina.

Como funciona: A função carregarDados simula uma espera çıra de 2 segundos com setTimeout e chama o callback com dados fictícios. O resultado aparece no <div id="output"> após a espera.

                
function carregarDados(callback) {
    setTimeout(() => {
        const dados = { id: 1, nome: "João" };
        callback(dados); // Executa após 2s
    }, 2000);
}

carregarDados(function(dados) {
    let output = document.getElementById("output");
    output.innerHTML = `Dados: ID: ${dados.id}, Nome: ${dados.nome}`;
});

document.getElementById("output").innerHTML = "Aguarde 2s...";

Promises

Promises são objetos que representam um valor futuro, com estados de sucesso ou erro.

Como funciona: A Promise simula uma tarefa que pode ter sucesso ou falhar. O método then lida com o sucesso, catch com erros, e finally sempre executa. O resultado aparece no <div id="output">.

                
function carregarDados() {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            const sucesso = Math.random() > 0.5;
            if (sucesso) {
                resolve({ id: 1, nome: "João" });
            } else {
                reject(new Error("Falha ao carregar"));
            }
        }, 2000);
    });
}

carregarDados()
    .then(dados => {
        let output = document.getElementById("output");
        output.innerHTML = `Dados: ${dados.id}, ${dados.nome}`;
    })
    .catch(erro => {
        let output = document.getElementById("output");
        output.innerHTML = `Erro: ${erro.message}`;
    })
    .finally(() => {
        let output = document.getElementById("output");
        output.innerHTML += `
Concluído`;
    });

document.getElementById("output").innerHTML = "Aguarde 2s...";

Async/Await

Async/await é uma forma mais clara de usar Promises, como se fosse código normal.

Como funciona: A função async usa await para esperar a Promise resolver. O resultado ou erro aparece no <div id="output"> após 1 segundo.

                
async function carregarDados() {
    return new Promise(resolve => {
        setTimeout(() => resolve({ id: 1, nome: "João" }), 1000);
    });
}

async function processar() {
    let output = document.getElementById("output");
    try {
        const dados = await carregarDados();
        output.innerHTML = `Dados: ${dados.id}, ${dados.nome}`;
    } catch (erro) {
        output.innerHTML = `Erro: ${erro.message}`;
    }
}

processar();
document.getElementById("output").innerHTML = "Aguarde 1s...";
