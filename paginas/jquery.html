O que é jQuery?

jQuery é uma biblioteca JavaScript rápida, leve e rica em recursos, projetada para simplificar a manipulação do DOM, o manejo de eventos, animações e requisições AJAX em páginas web. Seu lema é "escreva menos, faça mais".

Características Principais:

  • Seletores poderosos: Encontre elementos HTML facilmente
  • Manipulação do DOM: Altere conteúdo e estrutura com simplicidade
  • Gestão de eventos: Lide com cliques, teclas e mais
  • Efeitos e animações: Adicione transições suaves
  • AJAX simplificado: Carregue dados sem recarregar a página
  • Cross-browser: Funciona consistentemente em todos os navegadores

Por que usar jQuery?

  • Sintaxe simples e intuitiva
  • Reduz código JavaScript nativo complexo
  • Grande comunidade e plugins disponíveis
  • Compatível com projetos web modernos

Empresas como WordPress, IBM e Microsoft já usaram jQuery em projetos, embora o JavaScript moderno (ES6+) tenha substituído algumas de suas funções.

Instalação e Setup

Como Incluir jQuery:

  • CDN (recomendado): Carregue diretamente de servidores como jQuery CDN
  • Download: Baixe o arquivo e hospede localmente
  • NPM: Instale para projetos com build tools

Usando CDN:

                
// Adicione no  do HTML


// Verifique se jQuery está carregado
$(document).ready(function() {
    console.log('jQuery versão: ' + jQuery.fn.jquery);
    // Exibe: jQuery versão: 3.6.0
});

Download Local:

                
// Baixe de jquery.com e inclua


// Teste de carregamento
$(function() {
    alert('jQuery está pronto!');
});

NPM para Projetos Modernos:

                
# Instalar via NPM
npm install jquery

// Importar em seu JavaScript
const $ = require('jquery');

// Usar após carregar
$(document).ready(function() {
    console.log('jQuery carregado via NPM!');
});

Dica: Use $(document).ready() para garantir que o DOM esteja carregado antes de executar o código jQuery.

Seletores jQuery

Seletores permitem encontrar e manipular elementos HTML com base em ID, classe, tipo, atributos e mais, usando uma sintaxe semelhante ao CSS.

Seletores Básicos:

                
// Seletor por ID
$('#meuId').css('color', 'blue'); // Elemento com id="meuId"

// Seletor por classe
$('.minhaClasse').hide(); // Todos os elementos com class="minhaClasse"

// Seletor por elemento
$('p').text('Novo texto!'); // Todos os parágrafos

// Seletor por atributo
$('input[type="text"]').val('Digite aqui'); // Inputs do tipo texto

Seletores Avançados:

                
// Combinações
$('div.minhaClasse').css('background', 'yellow'); // Divs com classe específica

// Filhos e descendentes
$('#container > p').css('font-weight', 'bold'); // Filhos diretos
$('#container p').css('margin', '10px'); // Todos os descendentes

// Filtros
$('li:first').css('color', 'red'); // Primeiro item da lista
$('li:odd').css('background', 'lightgray'); // Items ímpares

Dica: Teste seletores no console do navegador para verificar quais elementos são afetados!

Eventos com jQuery

jQuery simplifica o manejo de eventos como cliques, teclas, mouse e formulários, com métodos fáceis de usar.

Eventos Comuns:

                
// Clique
$('#meuBotao').click(function() {
    alert('Botão clicado!');
});

// Mouseover e Mouseout
$('.elemento').hover(
    function() { $(this).css('background', 'yellow'); }, // mouseover
    function() { $(this).css('background', ''); } // mouseout
);

// Teclado
$('input').keyup(function(e) {
    console.log('Tecla pressionada: ' + e.key);
});

// Submissão de formulário
$('form').submit(function(e) {
    e.preventDefault(); // Impede o envio padrão
    console.log('Formulário submetido!');
});

Delegação de Eventos:

                
// Eventos em elementos dinâmicos
$('#container').on('click', 'button', function() {
    $(this).text('Clicado!');
});

// Múltiplos eventos
$('.item').on({
    click: function() { $(this).toggleClass('ativo'); },
    mouseenter: function() { $(this).css('color', 'blue'); },
    mouseleave: function() { $(this).css('color', ''); }
});

Dica: Use e.preventDefault() para evitar comportamentos padrão, como recarregar a página em formulários.

Efeitos e Animações

jQuery oferece métodos para criar efeitos visuais e animações, tornando as páginas mais interativas.

Efeitos Básicos:

                
// Esconder e mostrar
$('#elemento').hide(); // Esconde
$('#elemento').show(); // Mostra
$('#elemento').toggle(); // Alterna visibilidade

// Fade
$('#elemento').fadeOut(1000); // Desaparece em 1 segundo
$('#elemento').fadeIn(1000); // Aparece em 1 segundo
$('#elemento').fadeToggle(500); // Alterna com fade

// Slide
$('#elemento').slideUp(800); // Recolhe
$('#elemento').slideDown(800); // Expande
$('#elemento').slideToggle(800); // Alterna

Animações Personalizadas:

                
// Animação com propriedades CSS
$('#elemento').animate({
    width: '200px',
    opacity: 0.5,
    marginLeft: '50px'
}, 1000, function() {
    console.log('Animação concluída!');
});

// Encadeamento de efeitos
$('#elemento')
    .slideUp(500)
    .delay(1000)
    .slideDown(500)
    .css('background', 'lightblue');

Dica: O parâmetro de duração (em milissegundos) controla a velocidade, e a função de callback executa após o efeito.

AJAX com jQuery

AJAX permite carregar dados do servidor sem recarregar a página, e o jQuery simplifica isso com métodos poderosos.

Requisições Básicas:

                
// GET simples
$.get('https://api.exemplo.com/dados', function(data) {
    $('#resultado').text(JSON.stringify(data));
});

// POST simples
$.post('https://api.exemplo.com/enviar', { nome: 'João' }, function(data) {
    console.log('Resposta do servidor: ', data);
});

// Tratar erros
$.get('https://api.exemplo.com/dados')
    .done(function(data) {
        $('#resultado').html('Dados: ' + JSON.stringify(data));
    })
    .fail(function() {
        $('#resultado').html('Erro ao carregar dados!');
    });

Método $.ajax Completo:

                
// Requisição AJAX completa
$.ajax({
    url: 'https://api.exemplo.com/usuarios',
    method: 'POST',
    data: {
        nome: 'Maria',
        email: 'maria@email.com'
    },
    dataType: 'json',
    success: function(response) {
        $('#resultado').html('Sucesso: ' + response.message);
    },
    error: function(xhr, status, error) {
        $('#resultado').html('Erro: ' + error);
    },
    beforeSend: function() {
        $('#resultado').html('Carregando...');
    },
    complete: function() {
        console.log('Requisição concluída!');
    }
});

Dica: Use APIs públicas como jsonplaceholder.typicode.com para testar requisições AJAX com segurança.
