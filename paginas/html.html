O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Ela usa tags para estruturar conteúdo, como textos, imagens e links, permitindo que navegadores exibam páginas de forma organizada.

O HTML5, a versão mais recente, trouxe recursos como suporte a vídeos, formulários avançados e melhor acessibilidade. É fácil de aprender e essencial para desenvolvimento web.

Por que aprender HTML?

  • Base para criar sites
  • Compatível com todos os navegadores modernos
  • Suporta interatividade com CSS e JavaScript

Tags Principais

As tags HTML são blocos de construção que definem a estrutura e o conteúdo de uma página. Aqui estão algumas das mais importantes:

Tags de Estrutura

  • <html>: Raiz do documento
  • <head>: Metadados (não visíveis)
  • <body>: Conteúdo visível

Tags de Conteúdo

  • <h1> a <h6>: Títulos
  • <p>: Parágrafos
  • <a>: Links
  • <img>: Imagens

Exemplo de Código

<h1>Meu Título</h1>
<p>Este é um parágrafo com um <a href="https://istec-porto.pt">link</a>.</p>
<img src="https://istec-porto.pt/wp-content/uploads/2020/01/beAsset-1logo.png" alt="Descrição da imagem">

Estrutura Básica

Todo documento HTML segue uma estrutura padrão para garantir que o navegador o interprete corretamente. Veja o exemplo abaixo:

<!DOCTYPE html>
<html lang="pt">
<head>
    <meta charset="UTF-8">
    <title>Minha Página</title>
</head>
<body>
    <h1>Bem-vindo!</h1>
    <p>Esta é uma página simples.</p>
</body>
</html>

Explicação:

  • <!DOCTYPE html>: Indica que é HTML5
  • lang="pt": Define o idioma como português
  • <meta charset="UTF-8">: Suporta caracteres especiais

Tags Semânticas

Tags semânticas dão significado ao conteúdo, melhorando acessibilidade e SEO. Exemplos incluem:

  • <header>: Cabeçalho
  • <nav>: Navegação
  • <main>: Conteúdo principal
  • <footer>: Rodapé

Exemplo Semântico

<header>
    <h1>Título do Site</h1>
    <nav>
        <ul>
            <li><a href="#home">Início</a></li>
        <ul>
    </nav>
</header>
<main>
    <article>
        <h2>Meu Artigo</h2>
        <p>Conteúdo aqui...</p>
    </article>
</main>
<footer>
    <p>© 2024 Meu Site</p>
</footer>

Tabelas HTML

As tabelas HTML são usadas para exibir dados em formato tabular, como listas ou relatórios. Elas utilizam tags específicas para estruturar linhas e colunas.

Tags de Tabelas

  • <table>: Define a tabela
  • <tr>: Define uma linha
  • <th>: Define um cabeçalho de coluna
  • <td>: Define uma célula de dados

Exemplo de Tabela

<table border="1">
    <tr>
        <th>Nome</th>
        <th>Idade</th>
    </tr>
    <tr>
        <td>João</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Maria</td>
        <td>30</td>
    </tr>
</table>

Dica: Use o atributo border="1" para visualizar as bordas da tabela. Para estilização avançada, use CSS.

Formulários HTML

Formulários coletam dados dos usuários, como nomes ou emails. O HTML5 adicionou novos tipos de input para facilitar a validação.

Exemplo de Formulário

<form>
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" required>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    <button type="submit">Enviar</button>
</form>

Dica: Use required para tornar campos obrigatórios.

Elementos Multimédia

HTML5 suporta áudio, vídeo e imagens responsivas sem plugins externos.

Exemplo de Vídeo

<video controls>
    <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4">
    Vídeo não suportado.
</video>

Exemplo de Imagem Responsiva

<picture>
    <source media="(min-width: 800px)" srcset="https://via.placeholder.com/800x400">
    <img src="https://via.placeholder.com/400x200" alt="Descrição">
</picture>
